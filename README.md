# 세특 프롬프트 생성기 — 사용 안내서

> 수행평가 원문을 붙여넣으면, AI(Claude·ChatGPT 등)에 붙여넣을 **고품질 세특 작성 프롬프트**를 자동으로 만들어 줍니다.

---

## 🚀 시작하는 법 (딱 한 가지)

**`index.html` 파일을 더블클릭하세요.**

설치가 필요 없습니다. 인터넷이 없어도 됩니다. 입력한 내용은 어디에도 전송되지 않습니다.

---

## 📋 사용 순서

### 1단계 — 기본 정보 입력

| 항목 | 설명 |
|---|---|
| 학년도 | 현재 학년도 선택 (기본값: 올해) |
| 학년 | 고1 / 고2 / 고3 선택 |
| 교과명 | 직접 입력 (예: 심화국어, 미적분, 물리학Ⅱ) |
| 교과군 | 교과명 입력 시 자동 추천됨. 다르면 직접 변경 |
| 학생명 | 필수 입력 |
| 학번 | 선택 사항. 없으면 "미확인"으로 처리됨 |
| 프롬프트 강도 | 기본값: 최상위권형 |

> 💡 **교육과정 자동 판단**: 학년도와 학년을 선택하면 2022 개정 / 2015 개정이 자동으로 결정됩니다.

---

### 2단계 — 수행평가 원문 입력

**방법 A — 직접 붙여넣기 (권장)**
수행평가지에서 학생 답변 부분을 복사해 원문 입력란에 붙여넣으세요.

**방법 B — docx 파일 불러오기**
점선 박스 영역에 docx 파일을 드래그하거나, 클릭해서 파일을 선택하세요.
- 추출 후 ⚠️ **반드시 원문을 눈으로 확인**하고 체크박스를 선택해야 생성 버튼이 활성화됩니다.
- 표·체크박스·이미지 안의 글자는 자동으로 추출되지 않을 수 있습니다.

> 💡 **학생명·학번 자동 탐지**: 원문에 "이름 | 홍길동", "학번: 30419" 형태가 있으면 자동으로 후보를 추출해 [적용] 버튼으로 빠르게 입력할 수 있습니다.

---

### 3단계 — 생성 버튼 클릭

**✨ 세특 프롬프트 생성** 버튼을 클릭하면 프롬프트가 완성됩니다.

버튼이 비활성화(회색)인 경우: 필수 항목(학년도·학년·교과명·교과군·학생명·원문 80자 이상)을 모두 입력했는지 확인하세요.

---

### 4단계 — 생성된 프롬프트 사용

- **[복사]** 버튼으로 클립보드에 복사
- **[txt 다운로드]** 버튼으로 파일 저장 (`학생명_교과명_세특프롬프트_날짜.txt`)
- 복사한 프롬프트를 **Claude, ChatGPT, Gemini** 등 AI 채팅에 그대로 붙여넣으면 AI가 세특을 작성합니다.

---

## ⚙️ 프롬프트 강도 선택 기준

| 강도 | 적합한 상황 |
|---|---|
| 📝 일반형 | 안정적인 교과 활동 기록, 과장 없이 사실 중심 |
| 🎯 학종강화형 | 학업역량·탐구역량·발전가능성을 뚜렷이 드러낼 필요가 있을 때 |
| ⭐ 최상위권형 | 상위권 대학 학종 대비, 지적 깊이와 성장 흐름 최대화 |

---

## 📏 원문 처리 모드

| 모드 | 조건 | 처리 방식 |
|---|---|---|
| 전체형 | 원문 5,000자 이하 | 원문 전체를 AI가 읽고 세특 작성 |
| 압축형 | 원문 5,000자 초과 | AI가 핵심 답변·성찰·분석 부분을 먼저 선별 후 작성 |

> 이 도구는 원문을 직접 자르지 않습니다. 압축형에서도 원문 전체를 AI에게 전달하고, AI가 스스로 핵심 부분을 선별하도록 지시합니다.

## ❓ 자주 묻는 질문

**Q. 인터넷이 없어도 되나요?**
네. 이 도구 자체는 완전 오프라인으로 동작합니다. 단, 생성된 프롬프트를 Claude·ChatGPT 등에 붙여넣는 단계에서는 인터넷이 필요합니다.

**Q. 입력한 학생 정보가 저장되나요?**
아니요. 모든 입력 내용은 브라우저 안에서만 처리되며, 서버로 전송되거나 파일로 저장되지 않습니다. 브라우저를 닫으면 사라집니다.

**Q. hwp, pdf 파일도 불러올 수 있나요?**
아니요. docx(.docx) 형식만 지원합니다. 한글(hwp)은 텍스트를 직접 복사해 붙여넣기 해주세요.

**Q. 생성된 프롬프트를 어디에 붙여넣나요?**
Claude(claude.ai), ChatGPT(chat.openai.com), Gemini 등 AI 채팅 서비스의 입력창에 그대로 붙여넣으시면 됩니다.

**Q. 세특이 마음에 들지 않으면 어떻게 하나요?**
AI 채팅창에서 "700바이트 버전을 더 구체적으로 수정해줘" 등 추가 요청을 입력하면 됩니다.

```

├── index.html          ← 여기를 더블클릭해서 시작
├── README.md           ← 이 안내서

│   └── style.css
├── js/
│   ├── app.js
│   ├── prompt.js
│   ├── presets.js      ← 교과 프리셋·금지어 등 수정 가능
│   ├── curriculum.js
│   ├── extractor.js
│   ├── validator.js
│   ├── parser.js
│   └── utils.js
```
