# 세특 프롬프트 생성기 — 사용 안내서

> 학생 자료를 붙여넣으면, AI(Claude·ChatGPT 등)에 붙여넣을 **고품질 세특/창체·행특 작성 프롬프트**를 자동으로 만들어 줍니다.

---

## 🚀 시작하는 법 (딱 한 가지)

**`index.html` 파일을 더블클릭하세요.**

설치가 필요 없습니다. 인터넷이 없어도 됩니다. 입력한 내용은 어디에도 전송되지 않습니다.

---

## 🧭 두 가지 모드

화면 상단에서 용도에 맞는 모드를 먼저 선택하세요.

| 모드 | 대상 | 다루는 항목 |
|---|---|---|
| 👨‍🏫 담임용 | 담임교사 | 자율활동 / 진로활동 / 동아리활동 / 행동특성 및 종합의견 |
| 📚 교과용 | 교과교사 | 교과 세부능력 및 특기사항(세특) |

두 모드는 별도의 화면과 별도의 프롬프트 엔진을 사용하며, 최종 결과 확인·복사·다운로드는 같은 하단 영역을 공유합니다.

---

## 📋 교과용 사용 순서

### 1단계 — 기본 정보 입력

| 항목 | 설명 |
|---|---|
| 학년도 | 현재 학년도 선택 (기본값: 올해) |
| 학년 | 고1 / 고2 / 고3 선택 |
| 교과명 | 직접 입력 (예: 심화국어, 미적분, 물리학Ⅱ) |
| 교과군 | 교과명 입력 시 자동 추천됨. 다르면 직접 변경 |
| 학생 식별명 | 필수 입력. **실명 대신 "1번", "3반 15번" 같은 식별자 사용을 권장**합니다. 프롬프트가 외부 AI 서비스로 전송되기 때문입니다. |
| 학번 | 선택 사항. 없으면 "미확인"으로 처리됨 |
| 프롬프트 강도 | 사실중심형 / 균형형(기본값) / 심화형 |
| 교사 관찰 메모 | **선택이지만 강력 권장**. 입력하면 1순위 근거로 반영되고, 비워두면 원문만으로 보수적으로 작성됩니다. |
| 다중 버전 출력 | 기본은 700바이트 1개만 생성됩니다. 500/700/1000바이트 3버전이 모두 필요하면 체크하세요. |

> 💡 **교육과정 자동 판단**: 학년도와 학년을 선택하면 2022 개정 / 2015 개정이 자동으로 결정됩니다.

### 2단계 — 수행평가 원문 입력

**방법 A — 직접 붙여넣기 (권장)** / **방법 B — docx 파일 불러오기**(추출 후 반드시 원문을 확인하고 체크박스를 선택해야 생성 버튼이 활성화됩니다)

### 3단계 — 생성 → 복사/다운로드

---

## 👨‍🏫 담임용(창체·행특) 사용 순서

1. **작성 항목** 선택 (행동특성및종합의견 / 자율활동 / 진로활동 / 동아리활동)
2. **학생 식별명** 입력 (실명 대신 번호·약칭 권장), 목표 분량 선택
3. **교사 관찰 메모**(1순위 근거, 강력 권장) 및 **학생 자료**(자기평가서·상담자료·활동기록 등, 선택) 입력 — 최소 하나는 필수
4. 생성 → 복사/다운로드

담임용은 교과용과 달리 교육과정·교과군 판단이 없고, 항목별 관찰 포인트(예: 진로활동은 진로 탐색 과정, 동아리활동은 역할과 기여)가 자동으로 프롬프트에 반영됩니다.

---

## ⚙️ 프롬프트 강도 선택 기준 (교과용)

| 강도 | 적합한 상황 |
|---|---|
| 📝 사실중심형 | 안정적인 교과 활동 기록, 과장 없이 사실 중심 |
| 🎯 균형형(기본값) | 학업역량·탐구역량·발전가능성을 무리 없이 드러낼 때 |
| ⭐ 심화형 | 지적 깊이와 성장 흐름을 더 뚜렷이 드러낼 필요가 있을 때 |

---

## 📏 원문 처리 모드 (교과용)

| 모드 | 조건 | 처리 방식 |
|---|---|---|
| 전체형 | 원문 5,000자 이하 | 원문 전체를 AI가 읽고 세특 작성 |
| 압축형 | 원문 5,000자 초과 | AI가 핵심 답변·성찰·분석 부분을 먼저 선별 후 작성 |

> 이 도구는 원문을 직접 자르지 않습니다. 압축형에서도 원문 전체를 AI에게 전달하고, AI가 스스로 핵심 부분을 선별하도록 지시합니다.

## ❓ 자주 묻는 질문

**Q. 학생 이름을 그대로 써도 되나요?**
가능은 하지만 권장하지 않습니다. 생성된 프롬프트는 다음 단계에서 ChatGPT·Claude 등 외부 AI 서비스에 붙여넣게 되므로, 실명 대신 "1번", "A학생" 같은 식별자를 쓰는 것이 안전합니다.

**Q. 교사 관찰 메모를 꼭 입력해야 하나요?**
필수는 아니지만 강력히 권장합니다. 학교생활기록부 기재요령상 교사의 직접 관찰이 최우선 근거이며, 관찰 메모가 없으면 AI는 자료만으로 학생의 역량·태도를 단정하지 않고 보수적으로 작성합니다.

**Q. 인터넷이 없어도 되나요?**
네. 이 도구 자체는 완전 오프라인으로 동작합니다. 단, 생성된 프롬프트를 Claude·ChatGPT 등에 붙여넣는 단계에서는 인터넷이 필요합니다.

**Q. 입력한 학생 정보가 저장되나요?**
아니요. 모든 입력 내용은 브라우저 안에서만 처리되며, 서버로 전송되거나 파일로 저장되지 않습니다. 브라우저를 닫으면 사라집니다.

**Q. hwp, pdf 파일도 불러올 수 있나요?**
아니요. docx(.docx) 형식만 지원합니다. 한글(hwp)은 텍스트를 직접 복사해 붙여넣기 해주세요.

**Q. 세특이 마음에 들지 않으면 어떻게 하나요?**
AI 채팅창에서 "더 구체적으로 수정해줘" 등 추가 요청을 입력하면 됩니다.
