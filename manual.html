수행평가지에서 학생이 쓴 답변을 복사해 붙여넣으면, AI(Claude·ChatGPT 등)가 읽을 수 있는 고품질 세특 작성 프롬프트를 자동으로 만들어 줍니다.
교사가 직접 AI에게 긴 지시문을 작성할 필요가 없습니다.

Section 04
📄2단계 — 수행평가 원문 입력
A
방법 A — 텍스트 직접 붙여넣기 (권장)
수행평가지에서 학생 답변 부분을 Ctrl+C로 복사한 뒤, 원문 입력란을 클릭하고 Ctrl+V로 붙여넣습니다.
최소 80자 이상 입력해야 생성 버튼이 활성화됩니다.
💡 한글(hwp) 파일은 텍스트를 직접 복사해서 붙여넣기 해주세요. hwp 파일 자체를 업로드하는 기능은 지원하지 않습니다.
B
방법 B — docx 파일 불러오기
점선 박스 영역에 .docx 파일을 드래그하거나, 박스를 클릭해서 파일을 선택합니다.
텍스트가 자동으로 추출되면, 반드시 내용을 눈으로 확인하고 체크박스를 선택해야 생성 버튼이 활성화됩니다.
⚠️ 표·체크박스·이미지 안에 입력된 글자는 자동으로 추출되지 않을 수 있습니다. 추출 후 반드시 원문을 한 번 확인해 주세요.
🚫 doc·hwp·pdf 형식은 지원하지 않습니다. docx(.docx)만 가능합니다.
🤖 학생명·학번 자동 탐지: 원문에 "이름 | 홍길동", "학번: 30419" 형태가 있으면 [적용] 버튼이 나타나 빠르게 기본 정보를 채울 수 있습니다.

Section 06
🤖4단계 — AI에 붙여넣어 세특 받기
1
AI 채팅 서비스 열기
아래 서비스 중 하나를 브라우저에서 엽니다. 모두 무료 사용이 가능하지만, 유료 구독 시 더 좋은 결과를 얻습니다.
AI 서비스주소추천 모델
⭐ Claude (권장)claude.aiClaude Sonnet / Opus
ChatGPTchat.openai.comGPT-4o

2
프롬프트 붙여넣기
AI 채팅 입력창에 복사한 프롬프트를 Ctrl+V로 붙여넣고 전송하면, AI가 단계별 분석과 함께 500·700·1000바이트 세 버전의 세특을 자동으로 작성합니다.
3
결과 확인 및 수정 요청
AI가 생성한 세특이 마음에 들지 않으면, 같은 채팅창에서 추가 요청을 입력하면 됩니다.
추가 요청 예시
"700바이트 버전을 더 구체적으로 수정해줘"
"탐구 과정을 더 자세히 써줘"
"교사 문체가 너무 딱딱해. 좀 더 자연스럽게 다듬어줘"
"마지막 문장을 후속 탐구 질문으로 바꿔줘"
Section 07
⚙️프롬프트 강도 선택 기준
강도적합한 상황특징
📝 일반형 안정적인 교과 활동 기록, 과장 없이 사실 중심으로 정리할 때 간결하고 명확한 세특
🎯 학종강화형 학업역량·탐구역량·발전가능성을 뚜렷이 드러낼 필요가 있을 때 탐구 흐름과 성장 서술 강화
⭐ 최상위권형 상위권 대학 학종 대비, 지적 깊이와 성장 흐름을 최대화할 때 개념 연결·한계인식·후속탐구 포함
💡 확신이 없으면 최상위권형을 선택하세요. 기본값이며, 세특의 완성도가 가장 높습니다. AI가 출력 후 필요에 따라 축약하는 것이 역방향보다 쉽습니다.
Section 08
📏원문 처리 모드 (자동 적용)
모드조건처리 방식
전체형 원문 5,000자 이하 원문 전체를 AI가 읽고 세특을 작성합니다.
압축형 원문 5,000자 초과 AI가 핵심 답변·성찰·분석 부분을 먼저 선별 후 작성합니다.
ℹ️ 처리 모드는 원문 글자 수에 따라 자동으로 결정됩니다. 별도로 선택할 필요가 없습니다. 원문 입력란 하단에 현재 모드가 표시됩니다.
Section 09
❓자주 묻는 질문
인터넷이 없어도 사용할 수 있나요? ▼
이 도구 자체는 완전 오프라인으로 동작합니다. 단, 생성된 프롬프트를 Claude·ChatGPT 등에 붙여넣는 단계에서는 인터넷이 필요합니다.
입력한 학생 정보가 저장되거나 외부로 전송되나요? ▼
아니요. 모든 입력 내용은 브라우저 안에서만 처리되며, 서버로 전송되거나 파일로 저장되지 않습니다. 브라우저를 닫으면 모두 사라집니다.
hwp, pdf 파일도 불러올 수 있나요? ▼
아니요. docx(.docx) 형식만 지원합니다. 한글(hwp)은 내용을 직접 복사해 붙여넣기 해주세요. PDF는 내용을 선택해 복사할 수 있습니다.
생성 버튼이 계속 비활성화(회색)입니다. ▼
아래 항목을 모두 확인하세요.

① 학년도 선택 완료   ② 학년 선택 완료   ③ 교과명 입력 완료
④ 교과군 선택 완료   ⑤ 학생명 입력 완료   ⑥ 원문 80자 이상
⑦ docx 파일 사용 시 확인 체크박스 선택
세특이 마음에 들지 않으면 어떻게 하나요? ▼
AI 채팅창에서 추가 요청을 입력하면 됩니다. 예: "700바이트 버전을 더 구체적으로 수정해줘", "탐구 과정을 더 자세히 써줘" 등.
표 안에 입력된 학생 답변이 추출되지 않아요. ▼
docx 파일의 표 안에 입력된 텍스트는 추출 정확도가 낮을 수 있습니다. 이 경우 해당 부분을 직접 복사해 원문 입력란에 수동으로 추가해 주세요.
어떤 AI 서비스를 사용하는 것이 좋나요? ▼
이 프롬프트는 Claude(claude.ai)에 최적화되어 있습니다. ChatGPT, Gemini에서도 잘 동작하지만, 긴 지시문을 처리하는 품질은 Claude가 가장 우수합니다.
